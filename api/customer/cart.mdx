## Get Cart Items (Customer)

<api method="GET" path="/api/customers/cart-items/{customerId}" auth="Customer" />

Retrieves all items in the customer's shopping cart.

### Path Parameters

<ParamField path="customerId" type="string (uuid)" required>
  Customer identifier (must match authenticated user)
</ParamField>

### Response

<ResponseField name="items" type="array">
  Array of cart item objects
</ResponseField>

<ResponseField name="totalPrice" type="number">
  Total price of all items in cart
</ResponseField>

<ResponseField name="itemCount" type="integer">
  Total number of items in cart
</ResponseField>

### Response Codes

- `200 OK` - Cart items retrieved
- `404 Not Found` - Customer not found

### Examples

<CodeGroup>

```bash cURL
curl -X GET "https://your-server.com/api/customers/cart-items/customer-uuid" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  `https://your-server.com/api/customers/cart-items/${customerId}`,
  {
    headers: {
      'Authorization': `Bearer ${jwtToken}`
    }
  }
);

const cart = await response.json();
console.log(`Cart has ${cart.itemCount} items`);
console.log(`Total: $${cart.totalPrice}`);
```

```python Python
import requests

response = requests.get(
    f'https://your-server.com/api/customers/cart-items/{customer_id}',
    headers={'Authorization': f'Bearer {jwt_token}'}
)

cart = response.json()
print(f"Cart has {cart['itemCount']} items")
print(f"Total: ${cart['totalPrice']}")
```

</CodeGroup>

## Add Product to Cart (Customer)

<api method="POST" path="/api/customers/cart-items" auth="Customer" />

Adds a product to the shopping cart.

### Request Body

<ParamField body="customerId" type="string (uuid)" required>
  Customer identifier
</ParamField>

<ParamField body="productId" type="string (uuid)" required>
  Product to add to cart
</ParamField>

<ParamField body="quantity" type="integer" required>
  Quantity to add (must be > 0)
</ParamField>

<ParamField body="variantId" type="string (uuid)">
  Product variant ID (if product has variants)
</ParamField>

### Response Codes

- `204 No Content` - Product added to cart
- `404 Not Found` - Product, customer, or variant not found

### Examples

<CodeGroup>

```bash cURL
curl -X POST "https://your-server.com/api/customers/cart-items" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "customerId": "customer-uuid",
    "productId": "product-uuid",
    "quantity": 1,
    "variantId": null
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://your-server.com/api/customers/cart-items',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${jwtToken}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      customerId: customerId,
      productId: productId,
      quantity: 1,
      variantId: null
    })
  }
);

if (response.status === 204) {
  console.log('Product added to cart');
}
```

```python Python
import requests

response = requests.post(
    'https://your-server.com/api/customers/cart-items',
    headers={'Authorization': f'Bearer {jwt_token}'},
    json={
        'customerId': customer_id,
        'productId': product_id,
        'quantity': 1,
        'variantId': None
    }
)

if response.status_code == 204:
    print('Product added to cart')
```

</CodeGroup>

## Remove Item from Cart (Customer)

<api method="DELETE" path="/api/customers/cart-items/{customerId}/{cartItemId}" auth="Customer" />

Removes an item from the shopping cart.

### Path Parameters

<ParamField path="customerId" type="string (uuid)" required>
  Customer identifier
</ParamField>

<ParamField path="cartItemId" type="string (uuid)" required>
  Cart item identifier to remove
</ParamField>

### Response Codes

- `204 No Content` - Item removed from cart
- `404 Not Found` - Customer or cart item not found

### Examples

<CodeGroup>

```bash cURL
curl -X DELETE "https://your-server.com/api/customers/cart-items/customer-uuid/cart-item-uuid" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  `https://your-server.com/api/customers/cart-items/${customerId}/${cartItemId}`,
  {
    method: 'DELETE',
    headers: {
      'Authorization': `Bearer ${jwtToken}`
    }
  }
);

if (response.status === 204) {
  console.log('Item removed from cart');
}
```

```python Python
import requests

response = requests.delete(
    f'https://your-server.com/api/customers/cart-items/{customer_id}/{cart_item_id}',
    headers={'Authorization': f'Bearer {jwt_token}'}
)

if response.status_code == 204:
    print('Item removed from cart')
```

## Cart Management Tips

### Cart Behavior

<Info>
  **Duplicate Items**: Adding the same product increases the quantity of the existing cart item rather than creating a duplicate.
</Info>

<Info>
  **Persistent Cart**: Cart items are stored in the database and persist across sessions.
</Info>

<Warning>
  **Stock Availability**: Check stock availability before checkout. Items may sell out between adding to cart and completing purchase.
</Warning>

### Complete Cart Flow Example

```javascript
// 1. Add item to cart
async function addToCart(productId, quantity = 1, variantId = null) {
  const response = await fetch('/api/customers/cart-items', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      customerId,
      productId,
      quantity,
      variantId
    })
  });
  
  if (response.status === 204) {
    // Refresh cart display
    await loadCart();
  }
}

// 2. Load cart items
async function loadCart() {
  const response = await fetch(
    `/api/customers/cart-items/${customerId}`,
    {
      headers: { 'Authorization': `Bearer ${token}` }
    }
  );
  
  const cart = await response.json();
  displayCart(cart);
  updateCartBadge(cart.itemCount);
}

// 3. Remove item
async function removeFromCart(cartItemId) {
  const response = await fetch(
    `/api/customers/cart-items/${customerId}/${cartItemId}`,
    {
      method: 'DELETE',
      headers: { 'Authorization': `Bearer ${token}` }
    }
  );
  
  if (response.status === 204) {
    await loadCart();
  }
}

// 4. Proceed to checkout
async function checkout() {
  const cart = await loadCart();
  
  // Check stock availability
  const outOfStock = cart.items.filter(item => !item.inStock);
  if (outOfStock.length > 0) {
    alert('Some items are out of stock');
    return;
  }
  
  // Proceed to delivery selection and payment
  window.location.href = '/checkout';
}
```

### Updating Quantities

To update the quantity of an item in the cart:

1. **Remove** the existing cart item
2. **Add** the product again with the new quantity

Or implement quantity update UI:

```javascript
async function updateQuantity(cartItemId, newQuantity) {
  if (newQuantity <= 0) {
    await removeFromCart(cartItemId);
  } else {
    // Get product details from cart item
    const cart = await loadCart();
    const item = cart.items.find(i => i.id === cartItemId);
    
    // Remove old
    await removeFromCart(cartItemId);
    
    // Add with new quantity
    await addToCart(item.productId, newQuantity, item.variantId);
  }
}
```

### Cart Total Calculation

```javascript
function calculateCartTotal(cart) {
  // Calculate subtotal (with discounts applied)
  const subtotal = cart.items.reduce((sum, item) => {
    const price = item.discountPrice || item.price;
    return sum + (price * item.quantity);
  }, 0);
  
  // You can add delivery cost here if known
  // const total = subtotal + deliveryCost;
  
  return subtotal;
}
```
