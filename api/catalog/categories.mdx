## Get Parent Categories

<api method="GET" path="/api/categories/parent" auth="none" />

Retrieves all top-level (parent) categories.

### Query Parameters

<ParamField query="withCaching" type="boolean" default="true">
  Enable caching for faster response
</ParamField>

### Response

<ResponseField name="categories" type="array">
  Array of parent category objects
</ResponseField>

### Response Codes

- `200 OK` - Categories retrieved successfully

### Example

```bash cURL
curl -X GET "https://your-server.com/api/categories/parent"
```

### Success Response

```json
[
  {
    "id": "parent-1",
    "name": "Electronics",
    "iconUrl": "https://cdn.example.com/icons/electronics.svg",
    "childCount": 5
  },
  {
    "id": "parent-2",
    "name": "Clothing",
    "iconUrl": "https://cdn.example.com/icons/clothing.svg",
    "childCount": 8
  }
]
```

## Add Parent Category (Admin)

<api method="POST" path="/api/categories" auth="Admin" />

Creates a new parent category.

### Request Body (multipart/form-data)

<ParamField body="name" type="string" required>
  Category name
</ParamField>

<ParamField body="icon" type="file" required>
  Category icon image
</ParamField>

### Response Codes

- `204 No Content` - Category created
- `400 Bad Request` - Invalid data
- `409 Conflict` - Category name already exists

### Example

```bash cURL
curl -X POST "https://your-server.com/api/categories" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -F "name=Sports & Outdoors" \
  -F "icon=@/path/to/icon.svg"
```

## Add Child Category (Admin)

<api method="POST" path="/api/categories/{parentId}" auth="Admin" />

<ParamField path="parentId" type="string (uuid)" required>
  Parent category identifier
</ParamField>

<ParamField body="name" type="string" required>
  Child category name
</ParamField>

  Category image
</ParamField>

```bash cURL
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -F "name=Smartwatches" \
  -F "image=@/path/to/category-image.jpg"
```

## Delete Category (Admin)

<api method="DELETE" path="/api/categories/{categoryId}" auth="Admin" />

Deletes a category (parent or child).

### Path Parameters

<ParamField path="categoryId" type="string (uuid)" required>
  Category identifier
</ParamField>

### Response Codes

- `204 No Content` - Category deleted
- `404 Not Found` - Category not found

### Example

```bash cURL
curl -X DELETE "https://your-server.com/api/categories/category-id" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

<Warning>
  Deleting a parent category will also delete all its child categories and may affect existing products.
</Warning>

## Add Category Attribute (Admin)

<api method="POST" path="/api/categories/child/{childCategoryId}/attributes" auth="Admin" />

Adds a new attribute to a child category.

### Path Parameters

<ParamField path="childCategoryId" type="string (uuid)" required>
  Child category identifier
</ParamField>

### Request Body

<ParamField body="name" type="string" required>
  Attribute name (e.g., "Screen Size", "RAM")
</ParamField>

<ParamField body="required" type="boolean" required>
  Whether this attribute is required for products in this category
</ParamField>

### Response Codes

- `204 No Content` - Attribute added
- `400 Bad Request` - Invalid data
- `404 Not Found` - Category not found
- `409 Conflict` - Attribute already exists
